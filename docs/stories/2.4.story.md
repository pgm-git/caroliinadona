## Descricao

Implementar um visualizador de documentos PDF embutido na interface usando `react-pdf`, com suporte a zoom, rotacao, navegacao entre paginas e modo tela cheia. O visualizador sera usado na tela de detalhe do caso para permitir que usuarios consultem o documento original sem necessidade de download. Deve funcionar com signed URLs do Supabase Storage.

## Criterios de Aceite

- [ ] Visualizador de PDF renderiza documentos corretamente no browser
- [ ] Controles: zoom in/out (25%-200%), rotacao (90 graus), pagina anterior/proxima
- [ ] Indicador de pagina atual / total de paginas
- [ ] Navegacao por teclado: setas para paginas, +/- para zoom
- [ ] Loading state durante carregamento do PDF
- [ ] Tratamento de erro: exibir mensagem se PDF falhar ao carregar
- [ ] Modo tela cheia (fullscreen) com botao toggle
- [ ] Thumbnails das paginas na lateral (sidebar) para navegacao rapida
- [ ] Suporte a documentos de ate 100 paginas sem degradacao de performance
- [ ] Funciona com signed URLs do Supabase Storage (curta duracao)
- [ ] Responsive: visualizador se adapta ao container
- [ ] Preview de imagens (JPG/PNG/TIFF) como fallback simples (tag img)

## Notas de Implementacao

### Instalacao do react-pdf

```bash
npm install react-pdf
```

O `react-pdf` usa `pdfjs-dist` internamente. Configurar o worker:

```typescript
import { pdfjs } from "react-pdf";

pdfjs.GlobalWorkerOptions.workerSrc = new URL(
  "pdfjs-dist/build/pdf.worker.min.mjs",
  import.meta.url
).toString();
```

### Controles (PdfControls)

Toolbar com:
- Pagina: `< 1/10 >` com input numerico
- Zoom: `-` slider `+` com presets (50%, 75%, 100%, 125%, 150%)
- Rotacao: botao de rotacionar 90 graus
- Fullscreen: toggle
- Thumbnails: toggle sidebar

### Performance

- Usar `loading="lazy"` para paginas nao visiveis
- Renderizar thumbnails em resolucao baixa (scale=0.2)
- Implementar virtualizacao para documentos com muitas paginas (>20)
- Cache de paginas ja renderizadas

### Atalhos de Teclado

| Tecla | Acao |
|-------|------|
| `→` ou `Page Down` | Proxima pagina |
| `←` ou `Page Up` | Pagina anterior |
| `+` ou `=` | Zoom in |
| `-` | Zoom out |
| `0` | Reset zoom (100%) |
| `R` | Rotacionar 90 graus |
| `F` | Toggle fullscreen |
| `T` | Toggle thumbnails |
| `Home` | Primeira pagina |
| `End` | Ultima pagina |

### Tratamento de Erros

- URL expirada: exibir "Link expirado. Clique para recarregar." com botao de retry
- PDF corrompido: exibir "Nao foi possivel abrir o documento. O arquivo pode estar corrompido."
- Erro de rede: exibir "Erro ao carregar documento. Verifique sua conexao."
