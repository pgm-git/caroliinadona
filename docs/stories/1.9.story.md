## Descricao

Implementar o layout base da aplicação: sidebar de navegação colapsável, header com informações do usuário/organização, área de conteúdo principal e routing protegido. O layout segue o design system definido (Navy #1E3A5F como cor primária, Inter como fonte principal). Interface 100% em PT-BR.

## Criterios de Aceite

- [ ] Layout dashboard com sidebar esquerda + header superior + área de conteúdo
- [ ] Sidebar com navegação para: Dashboard, Casos, Documentos, Petições, Templates, Relatórios, Configurações
- [ ] Sidebar colapsável (toggle expand/collapse) com ícones + labels
- [ ] Sidebar destaca item ativo baseado na rota atual
- [ ] Header mostra: nome do usuário, role, nome da organização, avatar/iniciais
- [ ] Header com dropdown de usuário: Meu Perfil, Configurações, Sair
- [ ] Itens da sidebar filtrados por role (intern não vê Templates, Settings)
- [ ] Breadcrumb navigation no header
- [ ] Responsivo: sidebar se torna drawer em mobile (<768px)
- [ ] Loading state global (progress bar no topo durante navegações)
- [ ] Página 404 customizada em PT-BR
- [ ] Favicon e metadata PT-BR configurados

## Notas de Implementacao

- Design System:
  - Cor primária: Navy #1E3A5F (sidebar background)
  - Cor secundária: Blue #2563EB (links, botões)
  - Sucesso: Emerald #059669
  - Warning: Amber #D97706
  - Error: Red #DC2626
  - Font principal: Inter (sans-serif)
  - Font jurídica: Merriweather (para conteúdo de petições)
  - Font código: JetBrains Mono (para dados técnicos)
- Sidebar width: 256px expandida, 64px colapsada
- Usar `usePathname()` do Next.js para detectar rota ativa
- Itens de navegação:
  - Dashboard: `/dashboard` — ícone LayoutDashboard — todos os roles
  - Casos: `/cases` — ícone Briefcase — todos os roles
  - Documentos: `/documents` — ícone FileText — todos os roles
  - Petições: `/petitions` — ícone Scale — lawyer, coordinator, admin
  - Templates: `/templates` — ícone FileCode — coordinator, admin
  - Relatórios: `/reports` — ícone BarChart3 — coordinator, admin
  - Configurações: `/settings` — ícone Settings — admin
- Usar Zustand para estado da sidebar (collapsed/expanded) persistido em localStorage
- Breadcrumb gerado a partir dos segmentos da URL com labels em PT-BR
