# Story 3.5: Tela de revisao de extracao (side-by-side com confianca)

**Epic:** EPIC-03 Document Processing
**Prioridade:** P0
**Estimativa:** 8 pontos
**Status:** Draft
**Depende de:** 3.2, 2.4 (PDF viewer)
**Bloqueia:** 3.6

---

## Descricao

Implementar a tela de revisao onde o usuario visualiza o documento original (PDF/imagem) lado a lado com os dados extraidos. Cada campo extraido deve mostrar seu score de confianca com indicador visual colorido: verde (>= 80%), amarelo (60-79%), vermelho (< 60%). Campos com confianca abaixo de 80% devem ser sinalizados para revisao obrigatoria.

## Criterios de Aceite

- [ ] Pagina `/cases/[id]/documents/[docId]/review` com layout side-by-side
- [ ] Lado esquerdo: DocumentViewer (PDF/imagem) do story 2.4
- [ ] Lado direito: formulario com campos extraidos + badges de confianca
- [ ] Confianca >= 80%: badge verde
- [ ] Confianca 60-79%: badge amarelo

### Confidence Badge

```typescript
function getConfidenceColor(score: number) {
  if (score >= 80) return { bg: 'bg-green-50', text: 'text-green-700', border: 'border-green-200' };
  if (score >= 60) return { bg: 'bg-yellow-50', text: 'text-yellow-700', border: 'border-yellow-200' };
  return { bg: 'bg-red-50', text: 'text-red-700', border: 'border-red-200' };
}
```
