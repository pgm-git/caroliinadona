## Descricao

Implementar o pipeline de extracao de dados estruturados usando GPT-4o. O pipeline recebe o texto OCR de um documento e extrai campos estruturados especificos para cada tipo de documento (CCB, nota promissoria, contrato). Cada campo extraido deve ter um score de confianca individual. Os dados sao persistidos na tabela `extracted_data`.

## Criterios de Aceite

- [ ] Service `ExtractionService` em `src/server/services/extraction.service.ts`
- [ ] Prompts de extracao por tipo de documento (CCB, nota promissoria, contrato)
- [ ] Campos extraidos com score de confianca individual (0-100%)
- [ ] Schema JSON tipado para cada tipo de documento
- [ ] Dados persistidos na tabela `extracted_data` com `extractedFields` (JSONB)
- [ ] Confianca geral (`confidenceScore`) calculada como media dos campos
- [ ] Hash do prompt salvo em `extractionPromptHash` para rastreabilidade
- [ ] Modelo e versao salvos em `extractionModel` e `extractionVersion`
- [ ] `rawExtraction` salva a resposta bruta do GPT-4o
- [ ] Classificacao automatica de tipo de titulo (FR-015) com confianca >= 90%

## Notas de Implementacao

### Schema de Extracao (CCB)

```typescript
interface CcbExtraction {
  numeroCedula: { value: string; confidence: number };
  valorPrincipal: { value: number; confidence: number };
  taxaJuros: { value: number; confidence: number };
  indiceCorrecao: { value: string; confidence: number };
  dataVencimento: { value: string; confidence: number };
  dataEmissao: { value: string; confidence: number };
  devedor: {
    nome: { value: string; confidence: number };
    cpfCnpj: { value: string; confidence: number };
    endereco: { value: string; confidence: number };
  };
  avalista?: {
    nome: { value: string; confidence: number };
    cpfCnpj: { value: string; confidence: number };
  };
  credor: {
    nome: { value: string; confidence: number };
    cnpj: { value: string; confidence: number };
  };
}
```

### Prompt GPT-4o

```typescript
const CCB_EXTRACTION_PROMPT = `
Voce e um especialista em extracao de dados de Cedulas de Credito Bancario (CCB).
Analise o texto OCR abaixo e extraia os seguintes campos.
Para cada campo, forneca o valor extraido e um score de confianca de 0 a 100.
Se um campo nao for encontrado, retorne null com confianca 0.
Responda APENAS em JSON valido conforme o schema fornecido.
`;
```
